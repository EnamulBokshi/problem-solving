# Problem Solving Repository

This repository contains solutions to programming problems from various platforms (for example, Codeforces, LeetCode, and others). Solutions are written in multiple languages, primarily Python, C/C++, and TypeScript. The goal is to keep a clean, organized archive of practice problems and contests.

## Languages

- Python
- C / C++
- TypeScript

## Structure

The repository is organized by platform or topic. Each folder contains one or more problems with their corresponding solutions.

Common top-level folders include:

- `codeforces/` - Contest and practice solutions
- `leetcode/` - LeetCode problem solutions
- `ph-15-problems/` - Practice set solutions in TypeScript
- `basic/`, `random/`, `triangle-check/` - Miscellaneous practice problems

## How to Use

Browse to a folder, then open the solution file for the problem you want to review. Many problems include multiple solutions or languages in the same folder.

Example:

- `leetcode/` contains folders per problem
- `codeforces/` is split by contest type and round

## Notes

- Filenames and folder names follow the original problem or contest naming when possible.
- Some folders contain input/output samples or helper files used during practice.
